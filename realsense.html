Intel Realsense D435i

Description

The Intel RealSense D435i is an RGB-D depth camera with an integrated inertial measurement unit (IMU). These cameras provide robots with a 3 dimensional view of the world allowing you to create algorithms that reason about 3D structures. The addition of an IMU helps the system orient itself when moving.

Installation

There are two components required to use the RealSense from ROS: the library (librealsense) and the ROS wrapper (realsense-ros).

As these packages are not yet released for Ubuntu 20.04 or noetic I have created a custom .deb for you to install:

  1. Install the realsense library:

    curl -L https://drive.google.com/uc?id=1zPS8Q84a59-GDaZKQuR4KwtUS3zzoc56 > /tmp/nurealsense.deb
sudo apt install /tmp/nurealsense.deb

  2. Go to your nuws and clone the realsense ros package, and build the workspace:

    cd ~/nuws/src
git clone -b v2.2.17 https://github.com/IntelRealSense/realsense-ros.git
cd ../
catkin_make

You may now skip ahead to First Run.

First Run

  1. Plug in your Realsense. Make sure it is a USB 3 port
  2. run realsense-viewer
  3. Ignore the prompt to update the firmware
  4. If you turn on the RBG camera and stereo module you should be streaming 3D point clouds!

If you have problems let me know. I recommend against updating firmware at the moment but here is how to:

  1. On the left pane, click the More menu (hamburger menu) and select "Hardware Reset"
  2. Under the more menu select "Install Recommended firmware". Be patient, it takes a while.
  3. Restart the realsense-viewer. Under info you can view your firmware version (mine is 05.12.07.100)

Install From Source

STOP! You do not need these instructions. They are for reference, if you wish to install the realsense from source instead of from the binary package I have provided.

librealsense

Compile the library

git clone https://github.com/IntelRealSense/librealsense.git
cd librealsense
git checkout v2.38.1
cmake -B build -DCMAKE_BUILD_TYPE=RelWithDebInfo -DCMAKE_INSTALL_PREFIX=/usr -DBUILD_PYTHON_BINDINGS:bool=true .
cd build
make -j8

Create the installation script, called makeit and place it in build/

#!/usr/bin/env bash
cd ..
./scripts/setup_udev_rules.sh
cd build
make install
mv /usr/lib/python3/dist-packages/pyrealsense2/*.so /usr/lib/python3/dist-packages
mv /usr/lib/python3/dist-packages/pyrealsense2/*.so.* /usr/lib/python3/dist-packages

Make the script executable: chmod 755 makeit

Run checkinstall to create the package sudo checkinstall --pkgname=northwestern-librealsense ./makeit

librealsense-ros

mkdir -p ~/noetic/src
cd ~/noetic/src
git clone https://github.com/IntelRealSense/realsense-ros.git
# I use the development branch, commit e273f04
cd ..
rosdep install --from-paths src --ignore-src -r -y 
catkin_make -DCMAKE_BUILD_TYPE=RelWithDebInfo -DCMAKE_INSTALL_PREFIX=/opt/ros/noetic
cd build

Next create the makeit script

#!/usr/bin/env bash
source /opt/ros/noetic/setup.bash
make install

chmod 755 makeit

Finally, sudo checkinstall --pkgname=ros-northwestern-realsense-ros --exclude"opt/ros/noetic.rosinstall,/opt/ros/noetic/.catkin,/opt/ros/noetic/local_setup.bash,/opt/ros/noetic/local_setup.sh,/opt/ros/noetic/local_setup.zsh,/opt/ros/noetic/setup.bash,/opt/ros/noetic/setup.sh,/opt/ros/noetic/_setup_util.py,/opt/ros/noetic/setup.zsh" ./maket= Be sure to exclude the files in the home directory
