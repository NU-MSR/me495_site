PincherX 100

Overview

Each of you has a small 5-DOF PincherX 100 robot arm to use throughout your studies in this class. The arm has multiple control modes, but we will use ROS to program the arm.

Installation

  • The steps done in custom_ws should provide the necessary software
  • Install libnlopt-cxx0 libnlopt-cxx-dev libnlopt-dev swig
  • Please delete your nuws workspace and re-do those steps to gain access to the latest packages

Documentation

Software

Most of the software we sill use for the arm is documented with the source code for the robot here. As per the README.md, you should

  1. Read interbotix_descriptions/README.md to get a feel for the robot basics
  2. The low-level drivers are described by the interbotix_sdk
  3. The Gazebo Simulation is described in interbotix_gazebo
  4. The moveit interface is described in interbotix_moveit
  5. When using the software, the robot name is named px100

Source Code

  1. You are encouraged to look at the source code of the interbotix_ros_arms package to see how it works when you are unsure.
    • A good starting point is to start at a launchfile and work your way down into the included launchfiles, xacro files, and nodes
